Integration

Embed Snippet

Binde das Snippet in deine Website ein, um AI-Content automatisch zu laden.

Wie es funktioniert

Das Script verbindet deine Website mit dem seo.solutions Backend. Es läuft unsichtbar und ersetzt die markierten Elemente mit KI-optimiertem Inhalt.

JavaScript — in <head> einbinden 
<!-- seo.solutions AI Content Engine -->
<script>
(function() {
  var SEO_API_KEY = 'YOUR_API_KEY';
  var script = document.createElement('script');
  script.src = 'https://api.seo.solutions/v1/content/embed.js?key=' + SEO_API_KEY;
  script.async = true;
  document.head.appendChild(script);
})();
</script>

Zones im HTML markieren

HTML Beispiele 
<!-- Überschrift -->
<h1 data-seo-zone="main-headline">Titel hier</h1>

<!-- Fließtext -->
<p data-seo-zone="intro-text">Text hier...</p>

<!-- Bild Alt-Text -->
<img data-seo-zone="hero-img" src="..." alt="">

<!-- Meta Description -->
<meta name="description" data-seo-zone="meta-desc">
